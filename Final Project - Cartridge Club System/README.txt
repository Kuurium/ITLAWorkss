Instrucciones de Instalación de la Base de Datos:

Abra el archivo VideoGameClub_Script.sql en SQL Server y ejecútelo para crear la base de datos con datos de prueba.

Abra la solución en Visual Studio.

Vaya a la carpeta Data, archivo DatabaseHelper.cs.

Cambie la variable _connectionString poniendo el nombre de SU servidor SQL.

¡Ejecute el proyecto!
